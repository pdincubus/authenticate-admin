Reset your password

Before you start, you'll need:

  • Your email address
  • The mobile phone linked to your DWP Authenticate account

What will happen:

  1. You’ll enter the email address you normally use to sign in.
  2. We’ll send a password reset code to that email address.
  3. You’ll enter that password reset code.
  4. You’ll be authenticated by text or an app (if you’ve set that up).
  5. You’ll create a new password.
  6. You’ll be signed in to DWP Authenticate.
