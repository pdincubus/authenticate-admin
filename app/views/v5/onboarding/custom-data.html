Data passed by Authenticate

As a result of a successful user authentication, Authenticate will pass data for each user accessing your service.

The standard data is:

  • First name
  • Last name
  • Email address
  • Organisation name

Additional data can be customised beyond this. For instance this could be used to define any specific user roles or teams you may have within your service.
