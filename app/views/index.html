General changes

  • Added React single-pager apps for User list/Single user view/Email search results and organisations list researching. Added due to complexity required in user research testing scenarios, and lack of flexibility for this in prototype kit.
  • Added revised user list filter controls
  • Added a basic search - search by complete email address
  • Added a new journey for adding a user to multiple organisations
  • Increased prominence of user search box hint text due to some users not noticing it
  • Added an alternative journey for adding a user to multiple organisations
  • Added description about what access expired status means on L3 user details page
  • Added route to restore access to access expired status user on L3 user details page
  • Edited heading logo text to 'DWP Authenticate admin portal' to make it clearer what a the user is signing into
  • Removed superfluous content on the sign in pages based on the above change to the logo text in the header
  • Removed 'Invite expired' status after team review
  • Removed 'Signed out' page after team review
  • Added new your tasks card on the authoriser dashboard
  • Added new your tasks list for authorisers
  • Added new request review page for authorisers
  • Added revised add new user journey for requests/authorisations inc. new user metadata
  • Added revised add new organisation journey for requests/authorisations
